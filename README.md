## Jobs
- [Welcome to The Jungle](https://www.welcometothejungle.com/fr/jobs): Possibilité de filtrer les offres par localisation, secteur d'activités, type de contrat, niveau d'expérience, nombre de salariés, télétravail...
Sur chaque offre il est possible d'accéder à la page de l'entreprise avec une présentation de l'équipe et de la tech.
- [talent.io](https://www.talent.io/p/fr-fr/home): Création d'un profil sur le site puis les boîtes contactent les candidat.e.s.
- Les channels #jobs sur les slacks des communautés Ruby / Rails (voir plus bas)

## Meet-ups
Le site [meetup.com](https://www.meetup.com/) est une plateforme où il est possible de trouver des événements de réseautage par centres d'intérêt et selon sa localisation. Il y a beaucoup d'événements /conférences techs d'organisés, sur de nombreux sujets et pour tous niveaux. Certains sont passés en ligne avec la situation actuelle.

### À Paris (ou en ligne)
- [Mentor on Rails]()
- [Paris.rb](https://www.meetup.com/fr-FR/parisrb/): le premier mardi de chaque mois. N'importe qui peut [proposer un talk](https://rubyparis.org/talks) en respectant la ligne de conduite du meet-up.
- [Paris Ruby Workshop](https://www.meetup.com/fr-FR/Paris-Ruby-Workshop/): le deuxième mercredi de chaque mois. Possibilité de faire des katas avec d'autres développeur.se.s.

## Profil en ligne

En tant que développeur.se, montrer ce qu'on sait faire en ligne est important.

### GitHub

#### Organiser son GitHub
Si vous avez des projets sur votre GitHub, vous pouvez choisir ceux que vous voulez mettre en avant ou non (en [archivant](https://docs.github.com/en/github/creating-cloning-and-archiving-repositories/archiving-repositories) certains).

#### Page profil
Vous pouvez vous créer une page GitHub de [présentation de votre profil](https://docs.github.com/en/github/setting-up-and-managing-your-github-profile/managing-your-profile-readme).

- [GitHub Profile Readme Generator](https://arturssmirnovs.github.io/github-profile-readme-generator/): En répondant à quelques questions, cela permet de générer le code nécessaire à votre page profil.
- [Awesome Profile README templates](https://github.com/kautukkundan/Awesome-Profile-README-templates): En fouillant ce repo, vous trouverez de nombreux exemples de pages profils par catégorie pour vous inspirer.
- [Awesome Github Profile readme](https://github.com/abhisheknaiidu/awesome-github-profile-readme): Une autre liste de profils inspirants.
